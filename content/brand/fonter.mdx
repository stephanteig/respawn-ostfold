# Fonter

## Pixer (primær)

Primær pixel-font for all tekst unntatt Ø i logotypen.

- **Stil:** Pixel/bitmap
- **Bruk:** Titler, headings, logoer, display-tekst
- **Last ned:** [Fontfabric — Pixer](https://www.fontfabric.com/fonts/pixer/)
- **Lisens:** Gratis for personlig bruk

## Press Start 2P

Brukes **kun** for tegnet `Ø` i logotypen `RESPAWN ØSTFOLD`.

- **Stil:** Retro pixel
- **Bruk:** Ø-tegnet i logoer der Pixer mangler Ø
- **Last ned:** [Google Fonts — Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P)
- **Lisens:** SIL Open Font License

## Share Tech Mono (digital fallback)

Brukes på digitale flater der Pixer ikke er tilgjengelig (web, overlays, kontrollpanel).

- **Stil:** Monospace / teknisk
- **Bruk:** All web-tekst, UI-elementer, navigasjon
- **Last ned:** [Google Fonts — Share Tech Mono](https://fonts.google.com/specimen/Share+Tech+Mono)
- **Lisens:** SIL Open Font License

## Rajdhani (brødtekst)

Brukes for lengre brødtekst og lesbar body copy på nettsiden.

- **Stil:** Humanistisk sans-serif
- **Bruk:** Avsnitt og beskrivende tekst
- **Vekt:** 400, 600, 700

<Callout type="info">
  Overlayene bruker **Press Start 2P** fra Google Fonts, lastet automatisk via `@import`.
  Nettsiden bruker **Share Tech Mono** og **Rajdhani** via Google Fonts.
</Callout>
