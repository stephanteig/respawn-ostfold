# Deploy & Hosting

Respawn Østfold bruker to hosting-løsninger:

| Komponent | Platform | URL |
|-----------|----------|-----|
| Nettside + Docs + Overlays | GitHub Pages | `stephanteig.github.io/respawn-ostfold/` |
| WebSocket-server | Railway | `respawnostfold-production.up.railway.app` |

<Callout type="info">
  All deploy til GitHub Pages skjer automatisk via GitHub Actions
  når du pusher til `main`-branchen.
</Callout>
