# Railway WebSocket

WebSocket-serveren (`server.js`) kjører på Railway og er tilgjengelig på:

```
wss://respawnostfold-production.up.railway.app
```

## Helse-sjekk

```
https://respawnostfold-production.up.railway.app/health
```

Skal returnere: `Respawn Østfold WS Server OK`

## Miljøvariabler

Railway setter `PORT` automatisk. Serveren lytter på `process.env.PORT || 3001`.

## Tilkobling fra overlays

`_ws_client.js` velger automatisk riktig server basert på `location.hostname`:

```js
const WS_URL = (() => {
  if (location.hostname === 'localhost' || location.hostname === '127.0.0.1')
    return 'ws://localhost:3001';
  return 'wss://respawnostfold-production.up.railway.app';
})();
```

## Tilkobling fra kontrollpanelet

`app/kontrollpanel/page.tsx` gjør det samme:

```ts
const WS_BASE =
  typeof window !== 'undefined' &&
  (window.location.hostname === 'localhost' || window.location.hostname === '127.0.0.1')
    ? 'ws://localhost:3001'
    : 'wss://respawnostfold-production.up.railway.app';
```

<Callout>
  Railway free-plan-prosjektet `respawnostfold` er allerede satt opp.
  For å redeploy: push endringer til `Stream/respawn-overlays-v3/respawn-overlays/`
  og kjør `railway up` fra den mappen.
</Callout>
